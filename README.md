# Walltch

**Watch it all.** An addon-based media client for the desktop, built with Tauri.

Walltch is a neutral client: it does not host, provide or index any content.
It connects to addons the user chooses to install; what those addons serve is
the responsibility of their authors and users.

## Development

Requires [Rust](https://rustup.rs) and [Bun](https://bun.sh).

```bash
bun install
bunx tauri dev
```
